How It Works

  • The user registers (if not already registered) as one of the two roles â€“ Volunteer or Donor.

  • After successful authorization of the user the user is taken to the Home screen. This screen is common for all users.

  • From here the user can access all the screens.

  • The difference in working is in the forms for data entry, where the Donor can post donations whereas the volunteer can post requests and select the donations they need and schedule a pickup according to preference. Based on the role of the user the data is stored in different sections of the database and is fetched accordingly.

  • Another difference is in the profile section where the donor can manage his donations and see his past donations whereas the volunteer can control the status (Completed / In progress) of the donations they have received.
