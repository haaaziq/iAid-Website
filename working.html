Tools Used:

DART Programming Language

This application is developed using Dart as a programming language. Dart is a general-purpose programming language. Dart is an object-oriented programming language developed by Google. that means that every value in Dart is an object.

know more....

Flutter Framework

The UI of the application is developed using Flutter Framework. Flutter is an open-source mobile UI framework that helped developers build native-looking Android and iOS apps from the same code base. The central idea behind Flutter is the use of widgets.

know more....

Firebase

For authentication, storage, database collections (Backend) of this application we have used Firebase. Firebase is a Backend-as-a-Service (BaaS) app development platform that provides hosted backend services such as a real-time database, cloud storage, authentication, etc.

know more....

Gravit Designer

Initially, for designing the UI of our Application we have used Gravit Designer. Gravit Designer is a free full-featured vector graphic design app that works on ALL platforms.
